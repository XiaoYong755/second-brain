# LLM Index

## What This Area Is For

这里不是“LLM 资料收藏夹”，而是整个第二大脑里和 LLM 相关的长期知识地图。

目标不是把链接收得最多，而是把你真正会反复用到、反复思考、反复讲给别人听的内容沉淀成稳定页面。

## Start With

如果你今天只想往前推进一点点，优先做下面三件事之一：

- 读一条现有概念页，比如 [In-Context Learning](/Users/taobao/Desktop/第二大脑/10%20Wiki/LLM/Concepts/in-context-learning.md)
- 从一篇资料笔记里提炼一条长期知识页，比如 [Attention Is All You Need](/Users/taobao/Desktop/第二大脑/40%20Library/Papers/attention-is-all-you-need.md)
- 在 `Open Questions` 里记下一个你还没想透的问题

## Suggested Learning Path

如果你想让这个区域长得更扎实，可以按这个顺序补：

1. `Concepts`
2. `Systems`
3. `Models`
4. `Papers`
5. `Open Questions`

原因很简单：

- `Concepts` 让你知道核心术语到底在说什么
- `Systems` 让你把模型和工程现实接起来
- `Models` 让你知道今天谁在做什么
- `Papers` 让你保留源头
- `Open Questions` 让这张图谱继续活着

## Promotion Rule

这里最重要的一条规则是：

- `Library` 记录你看到了什么
- `Wiki` 记录你现在怎么理解它

也就是说，一篇论文笔记不等于一条长期知识页。只有当某条内容已经从“摘录”变成“理解”，才值得提炼进这里。

## Keep It Alive

一个好用的 LLM 区域，不是靠一次搭完，而是靠反复做这三件事：

- 发现一个重要问题
- 写下一条更清楚的解释
- 把相关页面重新链接起来
