# Second Brain Bootstrap

## Goal

Build a local Markdown second brain that supports LLM learning, personal knowledge, and project execution without requiring any specific app.

## Current status

The first scaffold is in place with folders, templates, friendly entry pages, GitHub sync, and the first batch of core LLM concept notes.

## Decisions

- Use a mixed structure instead of pure PARA
- Keep `Wiki` as the system core
- Use `Inbox` for fragmented capture
- Keep file names mostly in English and note bodies mixed-language
